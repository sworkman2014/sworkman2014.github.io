Peer Pulse

React.JS, Firebase, UI & UX

Peer Pulse is a polling app that allows peer-to-peer voting on shared ideas either publicly or privately.

Built in React.JS (and soon-to-be converted to React Native), Peer Pulse is a web application maximized for smartphones. The use of Firebase allows data to be updated in reltime, so you're never waiting for votes to refresh.


Wise Rides

Express, Node.JS, MongoDB

Wise Rides, a ride-scheduling app for the elderly, was developed along side two other developers. My role was working both with front and back-end technologies.   

Express was used to handle the front-end and Node.JS was used to make primarily get and post requests to MongoDB, which managed the data.


myReddit

UI / UX DESIGN, AJAX, jQuery

A simple re-design of the popular "front page of the internet," Reddit.com as a single-page web application.

AJAX was used to make get requests from Reddit's API along with jQuery to display updated information on various Reddit pages.
